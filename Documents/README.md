**----Procedurally Generated 3D Platformer Level----**

Created using Godot and C#.





**Instructions:**

1\) Open controls.png to read the game's controls.

2\) Open the exe file (Windows) or X86\_64 file (Linux) to play the game.

3\) Feel free to adjust the game's parameters to how you want it. Once adjusted, click "Start" to play (You can play with the default parameters instead if you wish).

3\) Play and complete the level.

4\) Complete the form. You will be required to upload a JSON file, which should now be in the same folder where the application is.
